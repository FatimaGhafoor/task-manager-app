# Task Manager App

A Task Management application built without any JavaScript framework — assigned as an interview task.

## Features

- Mock API login
- Fetch, add, edit, delete tasks
- Search (debounced), filter, sort, pagination
- Drag and drop between Todo / In Progress / Done
- LocalStorage fallback
- Dark/Light mode
- Responsive design

## Tech Decisions

- Mock API simulated using Promises + setTimeout (per "no framework" requirement)
- Vanilla JS with ES Modules, modular architecture
